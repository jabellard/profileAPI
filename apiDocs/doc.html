Profile API API documentation version 1.1.0

Headline

RESTful API with the following features:

  • JSON Web Token authentication/authorization
  • paging and filtering on collection-based resources, and
  • CRUD operations on item-based resources

Get all the profiles in the collection.

post

Query Parameters

  • firstName: (string)

    Query by first name.

    Example:

    sampleFirstName
  • lastName: (string)

    Query by last name.

    Example:

    sampleLastName
  • age: (integer)

    Query by age.

    Example:

    10
  • gender: (string)

    Query by gender.

    Example:

    M
  • id: (string)

    Query by id.

    Example:

    77777
  • page: (integer - default: 1)

    Page of results to return.

    Example:

    1
  • limit: (integer)

    Number of results per page.

    Example:

    10

Body

Media type: application/json

Type:

[object Object]
Properties
  • results: required (array of Profile)

    Items: Profile

    • firstName: required (string)
    • lastName: required (string)
    • age: required (integer - minimum: 1)
    • gender: required (one of M, m, F, f)
    • id: required (string - minLength: 5 - maxLength: 5 - pattern: ^\d+$)

    Example:

    {
  "firstName": "sampleFirstName",
  "lastName": "sampleLastName",
  "age": 30,
  "gender": "M",
  "id": "77777"
}
