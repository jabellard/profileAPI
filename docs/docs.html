Profile API documentation version 1.1.0

Headline

RESTful API with the following features:

  • JSON Web Token authentication/authorization
  • paging and filtering on collection-based resources, and
  • CRUD operations on item-based resources

/profiles

/profiles get post

get

Get all the profiles in the collection.

post

Create a new profile.

get /profiles

Get all the profiles in the collection.

Secured by jwt

JSON Web Token authentication/authorization security scheme.

  • Response

Headers

  • Authorization: required (string)

    Authorization header.

    Example:

    Bearer <jwt>

Query Parameters

  • firstName: (string)

    Query by first name.

    Example:

    sampleFirstName
  • lastName: (string)

    Query by last name.

    Example:

    sampleLastName
  • age: (integer)

    Query by age.

    Example:

    10
  • gender: (string)

    Query by gender.

    Example:

    M
  • id: (string)

    Query by id.

    Example:

    77777
  • page: (integer - default: 1)

    Page of results to return.

    Example:

    1
  • limit: (integer)

    Number of results per page.

    Example:

    10
